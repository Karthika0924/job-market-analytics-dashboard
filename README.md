# Job Market Analytics Dashboard

## Project Overview
This project is an interactive Power BI dashboard built to analyze job market trends, salary insights, hiring locations, and in-demand skills.

## Features
- Interactive filters (Location, Category, Job Type, Skills)
- Hiring trend analysis
- Salary analysis by category
- Top hiring locations
- Most demanded skills

## Tools Used
- Power BI
- Power Query
- Data Cleaning
- Data Visualization
